1 需求背景（必填）

当前模型精调已经支持SFT、DPO，这些微调方式进行的前提是已经具备了相关领域基础知识，通过监督微调和人类偏好对齐的方式使模型输出更符合业务预期。但如果模型缺少业务所需的通用知识，需要大量补充新知识的场景，万擎还未支持，无法支持相关用户诉求。同时，业界平台百度千帆、阿里百炼、讯飞星辰均已支持相关的能力，在能力不具备的情况下会损失商机。

基本信息

任务名称：必填，占位“请输入任务名称”，支持中英文、数字、下划线、中划线，不能以下划线和中划线开头，最长64字符

**任务类型：必填，下拉可选文本生成、图像理解，仅选择文本生成时，精调方式展示CPT，选择图像理解时，CPT选项不展示**

**任务描述：非必填，占位“请输入任务描述”，最长200字符**

**标签：非必填，支持添加标签，标签项目间隔**

**精调方式**

**新增CPT，若用户选择文本生成类型的任务，精调方式仍然默认选中SFT**

**训练配置**

**训练方法：全量更新，仅展示该选项并选中**

**参数配置：根据所选模型展示对应的参数配置，包含超参数名称、参数默认值（含参数可修改的上下限）、参数说明，其中迭代轮次建议默认为1，具体以后端提供的参数及默认值为准**

**数据配置**

**数据集：仅展示CPT类型的训练集，支持项目数据集和预置数据集，需后端提供预置数据集，支持多选**

**默认关闭，开启时展示以下字段**

**语料类型：通用类（默认）、垂直领域类**

**若选择垂直领域类，则需要选择领域类型：以下为示例，科技类、医疗类、教育类、新闻类、社交媒体类，具体以后端提供的数据集类型为准**
