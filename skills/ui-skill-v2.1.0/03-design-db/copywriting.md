# 文案规范 (Copywriting)

## 核心原则

- **简洁直接**：操作词汇不超过 4 个字
- **主动语态**：描述用户行为，而非系统行为
- **一致性**：相同场景用相同措辞，不混用

---

## 按钮文案

### 主操作按钮（Primary）
| 场景 | 正确用法 | 错误用法 |
| :--- | :--- | :--- |
| 新建资源 | `+ 新建XXX` | `创建新的XXX` / `点击新建` |
| 提交表单 | `确认` | `提交` / `保存并提交` |
| 确认弹窗 | `确认` | `好的` / `是` |

### 次操作按钮（Default / Link）
| 场景 | 正确用法 | 错误用法 |
| :--- | :--- | :--- |
| 返回 / 放弃 | `取消` | `返回` / `不了` |
| 表格操作 | `启用` / `停用` / `编辑` / `删除` | `进行启用` / `点击编辑` |
| 查看详情 | 点击名称跳转（链接样式） | 独立"查看"按钮 |
| 更多操作 | `更多`（下拉菜单）| `操作` / `其他` |

## 状态文案

| 状态 | 文案 | 组件 |
| :--- | :--- | :--- |
| 运行中 | `运行中` | `<Badge status="processing" />` + 文字 |
| 已完成 | `已完成` | `<Tag color="success">` |
| 已停止 | `已停止` | `<Tag color="error">` |
| 异常 | `异常` | `<Tag color="warning">` |
| 启动中 | `启动中` | `<Badge status="processing" color="#FFAA00" />` + 文字 |
| 禁用 | `已禁用` | `<Tag>（灰色）` |

## 空状态文案

| 场景 | 文案 |
| :--- | :--- |
| 无数据（首次） | `暂无数据` |
| 搜索无结果 | `未找到相关结果，请调整搜索条件` |
| 字段值为空 | `–`（颜色 `#BBBDBF`，**禁止留空**） |

## 弹窗文案规范

### 确认类弹窗（危险操作）
```
标题：确认删除？
内容：删除后数据无法恢复，请谨慎操作。
按钮：取消（左）/ 确认删除（右，danger 样式）
```

### 新建/编辑弹窗
```
标题：新建{资源名} / 编辑{资源名}
按钮：取消（左）/ 确认（右，Primary 样式）
```

## 页面标题 / 副标题规范

- **主标题**：使用名词，不加动词。例：`在线推理`、`模型精调`
- **副标题**：一句话说明核心价值，12px，`#898A8C`，≤30 字

示例：
```
主标题：在线推理
副标题：将模型快速部署为 API 服务，支持实时调用
```
